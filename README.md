# Fairytale First Wish

This repository contains the Fairytale First Wish web app.

How to run locally

Prerequisites

- Node.js (v16+ recommended) and npm installed. You can use nvm to manage Node versions: https://github.com/nvm-sh/nvm

Quick start

```sh
# Clone the repository (replace <YOUR_GIT_URL> with your repo URL)
git clone <YOUR_GIT_URL>
cd fairytale-first-wish

# Install dependencies
npm install

# Start the development server
npm run dev
```

Build and preview

```sh
# Build for production
npm run build

# Preview the production build locally
npm run preview
```

The development server typically runs on http://localhost:5173 by default.
